The following commands can be used on this site

Command

Example

Syntax

Explanation

/roll
/roll 5 diffiulty 6 True
/roll <num_dice> difficulty <diff> <specialty>
Mostly self explanatory, noting that <specialty> is True or False. Specialty and difficulty can be left off and default to False and "difficulty 6." Can be used at the end of a post.
/rolls
/rolls 3 @ 5 diffiulty 6 True
/roll <num_rolls> @ <num_dice> difficulty <diff> <specialty>
Runs the /roll command num_rolls times. Can be used at the end of a post.
Warning: this will roll until botching or finished and increases difficulty on failures automatically.
Tags
<b></b> etc
As HTML
This applies the permitted tags to your text. The allowed tags are b, i, em, strong, p, br, strike, ul, li. Please don't overuse line breaks, they can make scenes harder to read.
@storyteller
@storyteller I botched with 10 paradox.
@storyteller <message>
This flags the scene as needing ST attention. It does not appear as a post. Capitalization of storyteller doesn't matter.
#WP
"I resist the mental attack #WP"
#WP
If you include #WP in a post, it will spend a willpower point. If used on a /roll command roll, it will add a success. This does not work with /rolls.
#Q
"I'm doing some magick #Q2", "I meditate at the node #Q-3"
#Q<number>
If you include #Q<number> in a post, it will spend quintessence. Positive numbers remove quintessence and negative numbers regain quintessence.
#P
"I'm botched my coincidental effect #P1", "I spent a week without gaining paradox #P-1"
#P<number>
If you include #P<number> in a post, it will gain paradox. Positive numbers add paradox and negative numbers remove paradox.
#B/#L/#A
"I've been hit for #3L"
#B<number>/#L<number>/#A<number>
These tags cause the character to take Bashing, Lethal, or Aggravated damage respectively. (Healing coming soon)
