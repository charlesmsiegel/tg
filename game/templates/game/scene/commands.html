Available Commands

Scene Commands Reference

Command Example Syntax Explanation
/roll /roll 5 difficulty 6 True /roll <num_dice> difficulty <diff> <specialty> Mostly self explanatory, noting that <specialty> is True or False. Specialty and difficulty can be left off and default to False and "difficulty 6." Can be used at the end of a post.
/rolls /rolls 3 rolls @ 5 difficulty 6 True /roll <num_rolls> rolls @ <num_dice> difficulty <diff> <specialty> Runs the /roll command num_rolls times. Can be used at the end of a post.
Warning: this will roll until botching or finished and increases difficulty on failures automatically.
/extended /extended 5 target 10 difficulty 7 True /extended <num_dice> target <target_successes> difficulty <diff> <specialty> Extended roll that accumulates successes until reaching the target number. Keeps rolling until target is reached or a botch occurs. Shows each roll and cumulative total. Difficulty and specialty are optional (default: 6 and False).
Tags <b></b> etc As HTML This applies the permitted tags to your text. The allowed tags are b, i, em, strong, p, br, strike, ul, li. Please don't overuse line breaks, they can make scenes harder to read.
@storyteller @storyteller I botched with 10 paradox. @storyteller <message> This flags the scene as needing ST attention. It does not appear as a post. Capitalization of storyteller doesn't matter.
#WP "I resist the mental attack #WP" #WP If you include #WP in a post, it will spend a willpower point. If used on a /roll command roll, it will add a success. This does not work with /rolls.
#WP "Rested for two nights #WP-2" #WP<number> If you include #WP in a post, it will spend Willpower points. Positive numbers remove Willpower and negative numbers regain Willpower. This version does not impact /roll!
#Q "I'm doing some magick #Q2", "I meditate at the node #Q-3" #Q<number> If you include #Q<number> in a post, it will spend quintessence. Positive numbers remove quintessence and negative numbers regain quintessence.
#P "I'm botched my coincidental effect #P1", "I spent a week without gaining paradox #P-1" #P<number> If you include #P<number> in a post, it will gain paradox. Positive numbers add paradox and negative numbers remove paradox.
#B/#L/#A "I've been hit for #3L" #B<number>/#L<number>/#A<number> These tags cause the character to take Bashing, Lethal, or Aggravated damage respectively. (Healing coming soon)
