## Implementation Gaps Summary

### Missing Delete Views
All models except CharacterTemplate lack delete views (by design - deletion through UI not implemented).

### Missing List Views
Most character types lack dedicated list views (use generic index only).

### Unimplemented Gamelines
- **Mummy (MtR)**: Detail views only, no create/update/forms
- **Group Models**: Coterie, Circle, Conclave not implemented

### Missing Reference Model Views
- **Mage**: Sphere, MageFaction, Fellowship, Instrument, Practice, Paradigm, Tenet
- **Wraith**: WraithFaction, Guild, Arcanos, ShadowArchetype
- **Changeling**: Cantrip, Chimera
- **Demon**: DemonHouse, DemonFaction, ApocalypticFormTrait
