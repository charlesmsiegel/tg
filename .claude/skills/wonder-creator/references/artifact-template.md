A brass compass with strange symbols instead of cardinal directions. The needle points toward whatever the user concentrates upon.

## The Power

**Spheres:** Correspondence 3
**Effect:** Needle points toward any person, place, or object the user has a connection to (true name, possession, blood)
**Activation:** Hold compass, concentrate on target, spend 1 Quintessence
**Limitations:** Range limited to same continent; warded targets cause needle to spin wildly
