```markdown
# [Node Name]

**Rank:** [X] | **Size:** [Description] | **Ratio:** [X.X]

## Concept

*[1-2 paragraphs: what is this place, why Quintessence gathers here, physical description, mystical atmosphere]*

| Stat | Value |
|------|-------|
| **Rank** | [X] |
| **Size** | [Description] ([rating]) |
| **Ratio** | [X.X] ([rating]) |
| **Quintessence/Week** | [X] |
| **Tass/Week** | [X] |

### Resonance

| Trait | Rating |
|-------|--------|
| [Trait] | [X] |
| **Total** | [X] |

### Merits & Flaws

| Merit/Flaw | Rating | Notes |
|------------|--------|-------|
| [Name] | [+/-X] | [How it manifests] |
| **Net Cost** | [X] | |

### Reality Zone

| Practice | Rating | Effect |
|----------|--------|--------|
| [Practice] | [+X] | -X difficulty |
| [Practice] | [-X] | +X difficulty |
| **Balance** | 0 | |

## Energy Manifestation
```

## Quick Stat Line

For brief references:
```
[Name]: Rank X | Size [desc] | Ratio X.X | Q:X T:X/week | [Key Merit/Flaw]
```

## Example

```markdown

**Rank:** 3 | **Size:** Small Building | **Ratio:** 0.5
```
