# Digital Web Hazards

## Intrusion Countermeasures (IC)

### White IC (Non-Lethal)

| IC Type | Effect | Typical Security |
|---------|--------|------------------|
| **Trace** | Tracks intruder's location | 1+ |
| **Alert** | Notifies administrator | 2+ |
| **Barrier** | Blocks access, requires bypass | 3+ |
| **Tar Pit** | Slows movement and actions | 3+ |
| **Scrambler** | Disrupts icon appearance | 4+ |
| **Maze** | Confuses navigation | 4+ |
| **Probe** | Scans for vulnerabilities | 5+ |
| **Blinder** | Obscures sector visibility | 5+ |
| **Crippler** | Temporarily disables abilities | 6+ |

### Gray IC (Damaging)

| IC Type | Effect | Typical Security |
|---------|--------|------------------|
| **Sparker** | Causes minor damage | 5+ |
| **Acid** | Degrades icon over time | 6+ |
| **Binder** | Immobilizes intruder | 6+ |
| **Ripper** | Tears at icon integrity | 7+ |
| **Psychotropic** | Causes confusion/fear | 7+ |
| **Killer** | Significant damage | 8+ |

### Black IC (Lethal)

| IC Type | Effect | Typical Security |
|---------|--------|------------------|
| **Black Hammer** | Attacks the mind directly | 9+ |
| **Flatline** | Attempts to kill user | 9+ |
| **Cascade** | Chain reaction damage | 10 |
| **Worm** | Follows user out of sector | 10 |

## Environmental Hazards

### Data Corruption

| Hazard | Description | Effect |
|--------|-------------|--------|
| **Noise** | Random data interference | -1 to all rolls |
| **Static** | Severe interference | -2 to all rolls, chance of disconnect |
| **Corruption Field** | Active data degradation | Damage to icon over time |
| **Null Zone** | Complete data absence | No abilities function |

### Instability

| Hazard | Description | Effect |
|--------|-------------|--------|
| **Lag Storm** | System instability | Random delays, whiteout risk |
| **Temporal Flux** | Time distortion | Unpredictable time passage |
| **Reality Glitch** | Local reality breakdown | Random effects, Paradox risk |
| **Crash Zone** | Imminent collapse | Must evacuate or be dumped |

### Traffic Hazards

| Hazard | Description | Effect |
|--------|-------------|--------|
| **Data Torrent** | Overwhelming information flow | Requires Navigation to traverse |
| **Spam Storm** | Junk data flood | Obscures everything, slows movement |
| **Packet Loss** | Data doesn't arrive | Actions may fail randomly |

| **Worm** | Spreading corruption | Moderate |
| **Trojan** | Disguised threat | Moderate-High |
| **Bot** | Automated attack drone | Low (in swarms: High) |
| **Hunter-Killer** | Targeted elimination program | High |

## Hazard Combinations

High-security Sectors often combine hazards:

**Corporate Standard (Security 6-7):**
- Trace + Alert + Barrier
- Probe + Sparker
- Binder backup

**Government Black Site (Security 8-9):**
- Maze + Blinder
- Ripper + Psychotropic
- Black Hammer failsafe
- Hunter-Killer patrols

**AI Core (Security 10):**
- Full IC suite
- Autonomous defenses
- Reality Glitch zones
- Flatline perimeter

## Detecting Hazards

| Method | Difficulty | Notes |
|--------|------------|-------|
| Passive scan | Awareness + Alertness | Only detects obvious hazards |
| Active probe | Intelligence + Computer | May trigger Alert IC |
| Spirit sight | Perception + Awareness | Detects entity presence |
| Pattern analysis | Intelligence + Technology | Identifies IC types |

## Bypassing IC

| Method | Roll | Notes |
|--------|------|-------|
| Stealth | Dexterity + Stealth | Avoid triggering |
| Hacking | Intelligence + Computer | Disable or bypass |
| Brute force | Strength + Brawl (icon) | Destroy IC |
| Social engineering | Manipulation + Subterfuge | Trick IC |
| Magic | Arete | Sphere-appropriate effects |
