# Fallen Fera Module

Fallen Fera are corrupted Changing Breeds who serve the Wyrm.

**For full information**, see the "fallen" section in each Fera's reference file:
- `lookup.py changing-breeds.ananasi ananasi` → Antara
- `lookup.py changing-breeds.bastet bastet` → Histpah (Liar-Kings)
- `lookup.py changing-breeds.corax corax` → Buzzards
- `lookup.py changing-breeds.mokole mokole` → Mnetics
- `lookup.py changing-breeds.nuwisha nuwisha` → Bitter-Grins
- `lookup.py changing-breeds.ratkin ratkin` → Mad Destroyers
- `lookup.py changing-breeds.rokea rokea` → Balefire Sharks

**Quick reference** is also in `modules/changing-breed.md` under "Fallen Fera".

## Creating Fallen Fera

Use standard Fera creation from `modules/changing-breed.md`, then add:
1. Wyrm taint (always detectable by Sense Wyrm)
2. Corruption-specific traits from the "fallen" section
3. Derangements (common but not universal)
4. Changed spiritual patron (Wyrm-spirit replaces original)

## Redemption

Unlike Black Spiral Dancers (who walked the Spiral and cannot return), Fallen Fera may theoretically find redemption:
- Requires genuine desire to change
- Extensive cleansing rites
- Original patron's forgiveness (rarely given)
- Taint never fully cleanses
- Most don't survive the attempt
