# Sept Module

Create septs (Garou communities) for W20.

## Sept vs Caern
- **Caern**: Sacred place (create first)
- **Sept**: Community protecting caern

## Leadership Positions

| Position | Role |
|----------|------|
| Alpha | Overall leader |
| Beta | Second in command |
| Warder | Caern security |
| Gatekeeper | Moon Bridge control |
| Master of the Rite | Ritual leader |
| Truthcatcher | Judges disputes |

## Sept Size

| Size | Garou | Kinfolk |
|------|-------|---------|
| Small | 5-15 | 20-50 |
| Medium | 15-30 | 50-150 |
| Large | 30-50 | 150-300 |
| Major | 50+ | 300+ |

## Output Format

```markdown
# [Sept Name]

**Size**: [Small/Medium/Large/Major]
**Caern**: [Link to caern document]
**Totem**: [Sept totem]

## Leadership
| Position | Name | Tribe | Rank |
|----------|------|-------|------|

## Population
[Garou count by tribe, Kinfolk description]

## Territory
[Bawn and extended claims]

## History
[Founding, major events]

## Current Situation
[Threats, goals, allies, enemies]
```
