# Haven Module

Create detailed havens — the sanctuaries where vampires sleep away the day.

## What is a Haven?

A haven is a vampire's daytime resting place, protected from sunlight and (ideally) intrusion. Haven quality directly impacts a vampire's security and comfort.

## Haven Rating Scale

| Rating | Description | Security | Examples |
|--------|-------------|----------|----------|
| ● | Minimal | Basic lock | Rented room, basement corner |
| ●● | Adequate | Good locks, some privacy | Apartment, small house |
| ●●● | Good | Solid security system | House with security, warehouse |
| ●●●● | Excellent | Professional security | Mansion, fortified building |
| ●●●●● | Exceptional | Near-impregnable | Underground complex, private compound |

## Creation Steps

1. **Location** — Where in the city
2. **Type** — Residential, commercial, unusual
3. **Rating** — Security and comfort level
4. **Security Measures** — Alarms, locks, guards
5. **Layout** — Rooms and spaces
6. **Access** — How to enter, escape routes
7. **Special Features** — Hidden rooms, panic rooms, etc.
8. **Vulnerabilities** — Weaknesses to address
9. **Document** — Write full description

## Haven Elements

### Location Considerations

| Factor | Details |
|--------|---------|
| Neighborhood | Crime rate, police presence, foot traffic |
| Accessibility | Public transit, parking, conspicuousness |
| Neighbors | Who might notice unusual hours? |
| Services | Power, water (some vampires skip these) |
| Legal ownership | Owned, rented, squatted, shell company? |

### Security Tiers

**Basic (Rating 1-2)**
- Quality locks
- Blackout curtains/shutters
- Hidden from casual observation

**Good (Rating 3)**
- Alarm system
- Reinforced doors
- Multiple locks
- Secured sleeping area

**Excellent (Rating 4)**
- Professional alarm with monitoring
- Security cameras
- Panic room/vault for sleeping
- Guards (mortal or ghoul)
- Multiple escape routes

**Exceptional (Rating 5)**
- Redundant systems
- Armed response
- Underground/hidden entrance
- Complete light-proofing throughout
- Self-destruct/purge capabilities

### Special Features

| Feature | Description |
|---------|-------------|
| Hidden room | Concealed sleeping chamber |
| Panic room | Reinforced safe room |
| Escape tunnel | Secret exit route |
| Blood storage | Refrigerated vitae supply |
| Guest quarters | Space for other vampires |
| Interrogation room | Soundproof, secure |
| Library | Research materials, occult texts |
| Workshop | For crafts, Vicissitude, etc. |
| Ritual space | For blood sorcery |
